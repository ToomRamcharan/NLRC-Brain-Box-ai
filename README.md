# NLRC AI Homework Solver

An intelligent homework solver application powered by Google's Gemini AI.

## Features
- **AI-Powered Solutions**: Get accurate answers to your homework questions.
- **Image Upload**: Upload images of your homework for analysis.
- **Secure API Key**: Uses a secure configuration for API keys.

## Setup
1. Open `index.html` in your browser.
2. The application will load with the configured API key.

## Security
- `config.js` contains the API key and is ignored by git.
- Do not commit `config.js` to version control.

## Technologies
- HTML5, CSS3, JavaScript
- Google Gemini API
